# TaleUnder
Projet académique de lycée en terminale NSI effectué avec d'autres camarades

## - Comment ouvrir les fichiers sources et jouer au jeu ?
- Télécharger la dernière release<br>
- Dézipper le fichier<br>
- Ouvrez le dossier 'TaleUnder-1.0.0'<br>
- Ouvrez le fichier main.py avec un IDE de code avec python et la dernière version de pygame installé<br>
- Cliquez sur le bouton play (en haut à droite de la fenetre pour VS Code)

Et voici le fichier source complet, et bon jeu !

## - Installer l'IDE VS Code, python et pygame
- Télécharger Visual Studio Code<br>
- Installez l'extension python et pip manager (onglet Extension à gauche)<br>
- Ouvrir l'onglet à gauche qui apparait (Pip Manager)<br>
- Appuyer sur le + (Add Python Package)<br>
- Dans la boite de dialogue qui s'affiche marquez 'pygame' et appuyez sur Entrée

Tout est installé ! 

## - Commandes de jeu :
- Déplacements Personnage et Menu :<br>
&ensp;- Haut : ↑<br>
&ensp;- Bas : ↓<br>
&ensp;- Droite : →<br>
&ensp;- Gauche : ←<br>

- Déplacement Spécial de combat :<br>
&ensp;- Coeur rouge : Déplacement libre (haut,bas,droite,gauche)<br>
&ensp;- Coeur bleu : Déplacement latéral libre et systeme de saut avec la touche ↑ enfoncé ou relaché<br>

- Menu choix de nom (début de jeu):<br>
&ensp;- Supprimer : Supp<br>
&ensp;- Valider : Entrée<br>

- Interaction Général : Z<br>
- Retour en arrière : Echap
